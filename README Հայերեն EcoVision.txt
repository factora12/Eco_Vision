 README (Հայերեն) EcoVision 
EcoVision — Իրական ժամանակում մեքենա և աղբի փոխազդեցության հայտնաբերում

EcoVision-ը իրական ժամանակում աշխատող համակարգ է, որը հիմնված է YOLOv9-ի վրա և հայտնաբերում է մեքենաներ և աղբ, հետևում է դրանց և պահպանում է սքրինշոթ, երբ աղբը մեքենայից հեռանում է՝ նախապես մոտ գտնվելուց հետո։

Օգտագործվում են՝

YOLOv9՝ օբյեկտների հայտնաբերման համար

Սեփական distance-based tracker (RadiusVectorTracker)

OpenCV՝ վիդեո մշակման համար

PyTorch՝ inference-ի համար

🧠 Ինչպես է աշխատում

Յուրաքանչյուր կադրում մոդելը հայտնաբերում է օբյեկտները։

Tracker-ը հետևում է դրանց՝ ըստ կենտրոնների հեռավորության։

Համակարգը հաշվարկում է աղբի և մեքենայի միջև հեռավորությունը։

Եթե աղբը գտնվում է NEAR_THRESHOLD պիքսելից պակաս հեռավորության վրա → համարվում է «մոտ»։

Երբ աղբը հեռանում է մեքենայից → պահվում է այն կադրը, որտեղ առաջին անգամ մոտ էր։

Բոլոր սքրինշոթները պահվում են output/ պանակում։
